# java-adv1

김영한의 **실전 자바 - 고급 1편, 멀티스레드와 동시성** 강의를 학습하며 실습한 예제 코드 저장소입니다.

## 학습 내용

- 멀티태스킹
- 멀티프로세싱
- 프로세스
- 스레드
- 스케줄링
- 컨텍스트스위칭

- Runnable

- 스레드생명주기: state 으로 확인
    - NEW
    - RUNNABLE
        - BLOCKED
        - WAITING
        - TIME WAITING
    - TERMINATED
- 스레드의 join
- 스레드의 interupt
    - volatile: 스레드 최신화정보 파악, boolean 정도만 적절 (변수의 변경 사항을 다른 스레드가 볼 수 있도록 가시성을 보장한다.)
    - 메모리 가시성: 한 스레드가 바꾼 변수 값을 다른 변수가 볼수있느냐 (스레드는 캐시메모리를 우선적으로 바라보기때문에 이러한 문제가 발생할수있는데 volatile을 선언하면 진짜 메모리를 보게됨)
- 스레드의 yield
- 메모리가시성: 한 스레드에서 변경한 데이터가 언제 다른 스레드에서 보이냐

- synchronized: 동시에 접근가능한 자원에 대해 한번에 하나의 스레드만 접근하도록 하는 메커니즘. 데이터 일관성 안정성 보장

## 프로젝트 구조

```text
src/
  thread/
    start/    # 스레드 시작 관련 예제
    control/  # 스레드 상태와 제어 예제
    join/     # join() 관련 예제
    util/     # 공통 유틸리티
```

## 실행 방법

IntelliJ IDEA에서 프로젝트를 열고, `src` 아래의 각 `Main` 클래스를 실행하면 예제를 확인할 수 있습니다.
