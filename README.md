# java-adv1

김영한의 **실전 자바 - 고급 1편, 멀티스레드와 동시성** 강의를 학습하며 실습한 예제 코드 저장소입니다.

## 학습 내용

- 스레드 생성과 실행
- `Runnable`과 `Thread` 사용
- 데몬 스레드
- 스레드 상태 확인
- `join()`을 활용한 스레드 제어
- 멀티스레드 환경에서의 기본 동작 이해

## 프로젝트 구조

```text
src/
  thread/
    start/    # 스레드 시작 관련 예제
    control/  # 스레드 상태와 제어 예제
    join/     # join() 관련 예제
    util/     # 공통 유틸리티
```

## 실행 방법

IntelliJ IDEA에서 프로젝트를 열고, `src` 아래의 각 `Main` 클래스를 실행하면 예제를 확인할 수 있습니다.
